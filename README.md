# MailPilot 🚀

Hello there,
MailPilot is a modern ASP.NET Core MVC web application designed to automate repetitive job application emails by allowing users to send professional emails with predefined templates and resume attachments to multiple recruiters or companies efficiently.

---

## 📌 Problem Statement

While applying for software development jobs, sending the same email content, subject, and resume attachment repeatedly to different HRs became time-consuming and inefficient.

MailPilot was built to solve this workflow by providing a reusable email automation system with template-based bulk mailing functionality.

---

## ✨ Features

- Bulk email sending
- Multiple recipient support
- Reusable email templates
- Automatic subject & body autofill
- Automatic resume attachment based on template
- Real-time email preview panel
- Modern SaaS-style UI
- File attachment support
- Gmail SMTP integration using MailKit
- SQL Server template storage
- Responsive dashboard layout

---

## 🛠 Technologies Used

- ASP.NET Core MVC (.NET 8)
- C#
- SQL Server
- Entity Framework Core
- MailKit
- Bootstrap 5
- JavaScript
- HTML5 & CSS3
